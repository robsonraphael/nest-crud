## Description

A basic crud using [Nest](https://github.com/nestjs/nest) framework

## Installation

```bash
$ npm install
```
or
```bash
$ yarn install
```
## Warning
In app.module.ts modify the snippet
```
const Conection = 'Insert key connect MongoDB-Atlas here !'
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```
